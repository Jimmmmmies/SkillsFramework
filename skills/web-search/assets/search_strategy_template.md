## Quick Reference

### Search Operators to Remember
- `"exact phrase"` - Exact match
- `site:example.com` - Site-specific
- `filetype:pdf` - File type
- `-exclude` - Exclude term
- `2020..2023` - Date range

### Reliability Indicators
- ✅ .gov, .edu domains
- ✅ Peer-reviewed publications
- ✅ Multiple source confirmation
- ✅ Recent updates
- ✅ Transparent authorship

### Red Flags
- ❌ No author/date information
- ❌ Excessive ads/sensationalism
- ❌ Single source for major claims
- ❌ Contradicts established knowledge
- ❌ Poor website quality
