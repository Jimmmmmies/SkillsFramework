# Source Evaluation Checklist

## Quick Assessment (30 seconds)
### First Impressions
- [ ] Website looks professional and well-maintained
- [ ] URL matches expected domain (.gov, .edu, .org, reputable .com)
- [ ] No excessive ads or pop-ups
- [ ] Clear navigation and organization

### Basic Information
- [ ] Author/creator clearly identified
- [ ] Publication/update date visible
- [ ] Contact information available
- [ ] "About Us" page exists and is informative

## Detailed Evaluation (2-3 minutes)

### Authority Assessment
**Author Credentials:**
- [ ] Author has relevant expertise/qualifications
- [ ] Author's affiliation is credible
- [ ] Author has other publications on the topic
- [ ] No obvious conflicts of interest

**Publisher/Organization:**
- [ ] Organization is reputable in the field
- [ ] Clear mission/purpose statement
- [ ] Editorial/quality control processes
- [ ] Transparency about funding/support

### Accuracy Assessment
**Evidence and Support:**
- [ ] Claims are supported by evidence
- [ ] Sources are cited/referenced
- [ ] Data/statistics are explained
- [ ] Methodology is described (if research)

**Fact-Checking:**
- [ ] Information aligns with other reliable sources
- [ ] No obvious factual errors
- [ ] Corrections/updates noted if needed
- [ ] References are current and relevant

### Currency Assessment
**Timeliness:**
- [ ] Publication date is appropriate for topic
- [ ] Information has been updated if needed
- [ ] Links are functional and current
- [ ] Reflects latest developments in field

**Historical Context:**
- [ ] Historical information is properly contextualized
- [ ] Older sources are identified as such
- [ ] Updates note changes from previous versions

### Purpose Assessment
**Intent and Bias:**
- [ ] Purpose is clear (inform, educate, sell, persuade)
- [ ] Balanced presentation of information
- [ ] Acknowledgment of limitations
- [ ] Transparency about perspective/bias

**Audience and Level:**
- [ ] Appropriate for intended audience
- [ ] Not overly simplistic or technical
- [ ] Clear about assumptions/prerequisites

## Specialized Checks

### For Academic Sources
- [ ] Peer-reviewed publication
- [ ] Published in reputable journal
- [ ] Clear methodology section
- [ ] References to related work
- [ ] Disclosure of funding sources

### For News/Media Sources
- [ ] Clear distinction between news and opinion
- [ ] Multiple sources cited
- [ ] Corrections policy published
- [ ] Transparency about sources
- [ ] Professional editorial standards

### For Commercial Sources
- [ ] Clear about commercial intent
- [ ] Accurate product/service information
- [ ] Contact/support information
- [ ] Customer reviews available elsewhere
- [ ] Compliance information provided

### For Government/Organization Sources
- [ ] Official domain (.gov, .edu, .org)
- [ ] Clear organizational structure
- [ ] Regular updates/maintenance
- [ ] Transparency about data collection
- [ ] Accessibility standards met

## Red Flag Detection

### Critical Red Flags (Avoid Source)
- [ ] No author/date information
- [ ] Contradicts established scientific consensus
- [ ] Promotes harmful/dangerous information
- [ ] Clear malicious intent
- [ ] Plagiarism or copyright violation

### Warning Signs (Use with Caution)
- [ ] Excessive emotional language
- [ ] Sensational headlines
- [ ] Anonymous sources without justification
- [ ] Poor website design/functionality
- [ ] Limited/about page information

### Minor Concerns (Verify Independently)
- [ ] Minor grammatical/spelling errors
- [ ] Outdated design but current content
- [ ] Limited scope/perspective
- [ ] Self-published without peer review

## Cross-Verification Steps

### Multi-Source Confirmation
- [ ] Check 2-3 other reliable sources
- [ ] Look for consensus among experts
- [ ] Verify statistics with original data
- [ ] Check for recent developments

### Context Verification
- [ ] Understand broader topic context
- [ ] Check historical background if needed
- [ ] Consider cultural/political context
- [ ] Note any controversies in the field

## Final Decision Matrix

### Score Your Source
Give 1 point for each checked item in Detailed Evaluation (excluding Red Flags)

**Scoring:**
- 20+ points: **Excellent Source** - Highly reliable, can be trusted
- 15-19 points: **Good Source** - Generally reliable, minor concerns
- 10-14 points: **Fair Source** - Use with caution, verify key claims
- Below 10 points: **Poor Source** - Avoid or use only for background

### Decision Guidelines
- **Cite with confidence**: 20+ points, no critical red flags
- **Cite with caution**: 15-19 points, note any limitations
- **Use for context only**: 10-14 points, don't rely on key claims
- **Avoid citing**: Below 10 points or has critical red flags

## Quick Reference Guide

### Trust Indicators
- ✅ .gov, .edu domains
- ✅ Peer-reviewed publications
- ✅ Multiple source confirmation
- ✅ Transparent authorship
- ✅ Recent updates
- ✅ Professional presentation

### Verification Steps
1. Check author credentials
2. Verify publication date
3. Cross-reference with other sources
4. Evaluate supporting evidence
5. Consider purpose and bias

### When in Doubt
1. Don't use the source for key claims
2. Find alternative reliable sources
3. Note the uncertainty in your work
4. Seek expert opinion if needed
