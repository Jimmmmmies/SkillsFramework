# Search Operators and Advanced Techniques

## Google Search Operators (Complete Reference)

### Basic Operators

| Operator | Example | Purpose |
|----------|---------|---------|
| `"exact phrase"` | `"machine learning"` | Exact phrase match |
| `OR` or `|` | `python OR javascript` | Either term |
| `AND` or `+` | `python +tutorial` | Both terms required |
| `-` (minus) | `python -snake` | Exclude term |
| `*` (wildcard) | `best * for beginners` | Unknown word placeholder |
| `()` (grouping) | `(python OR java) tutorial` | Group terms |

### Site and Domain Operators

| Operator | Example | Purpose |
|----------|---------|---------|
| `site:` | `site:github.com python` | Search within site |
| `inurl:` | `inurl:blog python` | Search in URLs |
| `intitle:` | `intitle:tutorial python` | Search in page titles |
| `intext:` | `intext:"machine learning"` | Search in page text |
| `filetype:` | `filetype:pdf python` | Specific file type |
| `ext:` | `ext:pdf python` | File extension |

### Date and Time Operators

| Operator | Example | Purpose |
|----------|---------|---------|
| `before:` | `python before:2023` | Results before date |
| `after:` | `python after:2022` | Results after date |
| `daterange:` | `python daterange:20220101-20221231` | Date range |
| `..` (range) | `python 2020..2023` | Number/date range |

### Advanced Operators

| Operator | Example | Purpose |
|----------|---------|---------|
| `related:` | `related:github.com` | Related sites |
| `cache:` | `cache:example.com` | Cached version |
| `info:` | `info:example.com` | Information about site |
| `define:` | `define:algorithm` | Definitions |
| `weather:` | `weather:New York` | Weather information |
| `stocks:` | `stocks:AAPL` | Stock information |

### 1. Problem-Solution Pattern
```

```

### 2. Comparison Pattern
```

```

### 3. Learning Pattern

## Common Search Problems and Solutions

### Problem: Too Many Results
**Solutions:**
- Add more specific keywords
- Use phrase searches with quotes
- Add `-` to exclude irrelevant terms
- Use `site:` to limit to trusted domains
- Add date constraints

### Problem: Too Few Results
**Solutions:**
- Remove less important keywords
- Try synonyms or related terms
- Use wildcards for variations
- Broaden the search scope
- Try different search engines

### Problem: Outdated Information
**Solutions:**
- Use `after:` operator for date limits
- Sort by date (if available)
- Check for "last updated" dates on pages
- Look for more recent sources on same topic

### Problem: Irrelevant Results
**Solutions:**
- Review and refine search terms
- Check for ambiguous terms
- Use more precise terminology
- Exclude common irrelevant associations

## Search Quality Assessment

### Quick Evaluation Checklist
- [ ] Are results from authoritative sources?
- [ ] Is information current/relevant?
- [ ] Do multiple sources agree?
- [ ] Is there supporting evidence?
- [ ] Are sources properly cited?

### Red Flags in Search Results
- Results only from forums or social media
- Excessive ads or sponsored content
- Poor website design/functionality
- No author/date information
- Claims that seem too good to be true

Remember: Effective searching is a skill that improves with practice. The best searchers know not just what to search for, but how to search for it.
